Use the guide hub in four steps

  1. Start with the product routes and confirm that the item link is current.
  2. Use the spreadsheet guide to record colour, size, item ID and seller notes.
  3. Review every warehouse QC image and request a focused photo only when a key detail is missing.
  4. Finish with the shipping guide before submitting a parcel.

Pause when information is incomplete

Do not continue automatically when a link expires, the selected variant is unclear, warehouse images do not match or the route rejects the item type. Return to the relevant step, correct the record and only continue when the order can be checked.
